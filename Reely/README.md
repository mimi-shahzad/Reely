# Reely 

Reely is an iOS app developed as the final project for the mobile development bootcamp. It can currently search through a comprehensive list of films by just a single keyword

## Installation

Clone the project using git, or download it and unzip the project. 

```bash
git clone https://github.com/mimi-shahzad/Reely.git
```

## Usage

- Open the project in Xcode and use `Command + R` to run the app for yourself. 
- The app should load and the user is presented with a search bar and UI
- type your desired film in the searchbar and hit enter and watch the list populate. 
- Scroll to the bottom to load more films.
